## Overview

XpressPro FX is a fully enterprise-grade hybrid fintech forex broker
trading and investment platform. It provides real-time trading,
investment portfolio management, secure payment processing via
MoonPay and Coinbase, and a full-featured admin portal.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Runtime | Node.js 20.x |
| Language | TypeScript 5.x |
| Framework | Express.js |
| Database | PostgreSQL via Neon (Serverless) |
| ORM | Prisma |
| Frontend | React + Vite |
| Admin Portal | React + Vite |
| Deployment | Railway (RAILPACK) |
| CI/CD | GitHub Actions |
| Security | Helmet, CORS, CSRF, Rate Limiting |
| Logging | Pino |
| Payments | MoonPay, Coinbase Commerce |
